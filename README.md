# decastreamer
Web app for streaming visualization of Deca tag devices

## Installation

python -m pip install --upgrade pip

Python 3 required. Modules needed:
- pillow
- appjar
- matplotlib
- serial

http://appjar.info/pythonBasics/

New serial: POS,1,<name>,x,y,z,<confidence>,hex value
